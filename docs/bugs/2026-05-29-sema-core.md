## P1 — High

### [CORE-1] `&display[..39]` byte-slice panics on non-ASCII value display
- **Location:** `crates/sema-core/src/error.rs:310`
- **Confidence:** 95%
- **Why:** `type_error_with_value` slices the display string at a hard byte offset of 39. If the
  string is longer than 40 bytes and byte 39 falls inside a multi-byte char (any value whose
  display contains `λ`, emoji, etc.), Rust panics with "byte index 39 is not a char boundary".
  Reachable from any type error on a Unicode-containing value. (Pattern B.)
- **Fix:** Char-boundary-safe truncation, e.g.
  `display.char_indices().nth(39).map(|(i,_)| &display[..i]).unwrap_or(&display)`.

## P2 — Moderate

### [CORE-2] Every named `define` creates an Rc cycle (env → lambda → env)
- **Location:** `crates/sema-core/src/value.rs:2207` + `crates/sema-eval/src/special_forms.rs:397-411`
- **Confidence:** 92% (may be accepted design — flag for a decision)
- **Why:** `eval_define` builds a `Lambda` with `env: env.clone()` (sharing the same
  `Rc<RefCell<bindings>>`), then `env.set(name, lambda)` inserts the lambda back into that very
  map. The cycle `Rc<bindings> → Value::Lambda → Lambda.env.bindings → Rc<bindings>` has no
  `Weak`, so the bindings map never drops. Every top-level/REPL `define` leaks its scope's
  bindings for the process lifetime — unbounded growth in long REPL/notebook sessions.
- **Fix:** Use a `Weak` for the self-reference, or a post-define patch that removes the current
  scope's `Rc` from the captured env. (Design change — confirm intent first.)

## P3 — Low

### [CORE-3] `sandbox.check(Caps::NONE, ...)` always denies in a restricted sandbox
- **Location:** `crates/sema-core/src/sandbox.rs:143-151`
- **Confidence:** 82% (latent — no current caller passes `Caps::NONE`)
- **Why:** `Caps::contains(self, NONE)` is `(self.0 & 0) == 0` → always true, so `check` returns
  `PermissionDenied` whenever anything is denied, even though requesting zero capabilities should
  always succeed.
- **Fix:** Early-return `Ok(())` when `required == Caps::NONE`.
