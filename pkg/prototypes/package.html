sema/http

1.2.0 networking

HTTP client and server primitives for Sema

$ sema pkg install sema/http
Readme Versions 8 Dependencies 2

Usage

Add sema/http to your project:

(import sema/http)

Making Requests

Simple GET request:

(define resp (http/get "https://api.example.com/data"))
(println (:status resp))  ; => 200
(println (:body resp))    ; => "{...}"

POST with JSON body:

(define resp
  (http/post "https://api.example.com/items"
    {:headers {"Content-Type" "application/json"}
     :body (json/encode {:name "widget" :price 9.99})}))

Server

Start a minimal HTTP server:

(http/serve 8080
  (lambda (req)
    {:status 200
     :headers {"Content-Type" "text/plain"}
     :body "Hello from Sema!"}))

Middleware

Compose middleware as higher-order functions:

(define (with-logging handler)
  (lambda (req)
    (println (format "~a ~a" (:method req) (:path req)))
    (handler req)))

(http/serve 8080
  (with-logging
    (lambda (req)
      {:status 200 :body "OK"})))
